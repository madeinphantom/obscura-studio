# Marketing Psychology Analysis

When the user invokes this workflow:

1. Read the skill at `~/.agents/skills/marketing-psychology/SKILL.md`
2. Apply the relevant mental models and psychological principles to the user's current context
3. Provide specific, actionable recommendations using the frameworks from the skill

## Focus Areas

- Identify which models apply to the situation
- Explain the psychology behind recommendations
- Suggest ethical implementation strategies

## Model Categories Available

- **Foundational Thinking**: First Principles, Jobs to Be Done, Inversion, 80/20
- **Buyer Psychology**: Mere Exposure, Endowment Effect, Loss Aversion, Status-Quo Bias
- **Persuasion**: Reciprocity, Scarcity, Authority, Social Proof, Anchoring
- **Pricing**: Charm Pricing, Rule of 100, Good-Better-Best, Mental Accounting
- **Design**: Hick's Law, AIDA, BJ Fogg Model, EAST Framework
- **Growth**: Flywheel, Network Effects, Compounding, Switching Costs
